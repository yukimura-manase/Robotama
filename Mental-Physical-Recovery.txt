
< Mental-Physical-Rrcovery🔥 >

1. 回復することは、楽しむことと同じレベルで、優先すべき事項！

    => 回復すべきなのは、Mental & Physical🔥

    => パフォーマンスの発揮や、日々を楽しむために必要🔥

2. Physicalを回復すること

    => 一言でまとめると『生活習慣』🔥

    => 睡眠8h, 食事・栄養, 有酸素・筋トレ,

    => 各重要項目での工夫🔥

3. Mentalを回復すること

    => 心と向き合って、ちゃんと考えをまとめたり、答えを出していく。

    => 迷い・惑い・もやもやを解消する。

    => 日々を楽しむこと🔥


    3-1. 方向性とするとプラスを増幅させて、マイナスを除去する🔥

    => ストレスの低減 & ストレスの発散

    => 考え方(認識)の最適化・行動の最適化🔥

    => 楽しめることを増幅させていく🔥
