【CSS入門】floatの使い方を理解する

「floatは要素を浮かせる」ことが、floatプロパティの機能です。

floatは、指定した要素を浮動化（浮いた状態）にさせ、右や左に配置することができます。

横並びにする際に使用するfloatプロパティの値は、leftとrightの2つです。
要素を横並びにする際に、要素を左に寄せたいときはleft、右に寄せたいときはrightを使用します。

Block要素は、縦方向に積み上がる

Block要素を、横並びにするときに「 float 」が役立つ

[ float: left; ] Block

[ float: left; ]を設定すると、縦並びになっていた要素を、左側から順番に横並びにすることができます。

[ float: left; ]だと左側に横並びになる！

[ float: right; ] Block

[ float: right; ]を設定すると、縦並びになっていた要素を、右側から順番に横並びにすることができます。

上にある[ float: left; ] の影響を[ clear: left; ]で解除します

floatによる回り込みを解除するには、回り込む要素に対してclearプロパティを使います。
