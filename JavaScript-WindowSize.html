JavaScript で画面サイズ(幅・高さ)を変更を検知する・取得する方法

レスポンシブ対応のときに、ウィンドウサイズによって何かしらの処理を行うことがあります。 CSSではメディアクエリを使って、ウィンドウサイズによってスタイルを指定することができます。

JavaScriptではwindow.innerWidthでウィンドウ幅を取得して、判定しなければならず、CSSと違ってウィンドウサイズが変更されたらそれを検知して実行する必要があります。

resizeイベント設定すると端末のサイズ変更やスマホの傾きが検出された際のチェックが可能です。

端末サイズの変更のチェックは、resizeイベントを次のように記述することで実現できます。

ブラウザの表示可能領域の幅と高さは、window.innerWidth, window.innerHeightでそれぞれ取得できます。単位はピクセルです。

サイズを変更した時にブラウザサイズを参照したりすることで、メディアクエリのように画面幅に応じて処理を実行ししたりすることができます。

innerWidthとinnerHeightには、アドレスバーやスクロールバーなどの領域は含まれません。

  1. ウィンドウサイズの取得・判定をしてみよう
  2. JavaScript：端末サイズが変更されたことをチェックする(resizeイベント)
